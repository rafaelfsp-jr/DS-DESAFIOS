# Desafio: Bar OO

Repositório criado para versionar a solução do desafio **“Bar OO”** do curso DevSuperior.  
O objetivo é aplicar conceitos de **Programação Orientada a Objetos (POO)** em Java para calcular a conta de um cliente em um bar.

### Cenário

No bar:

- Ingresso:
  - Homens: R$ 10,00  
  - Mulheres: R$ 8,00  
- Consumo:
  - Cerveja: R$ 5,00  
  - Refrigerante: R$ 3,00  
  - Espetinho: R$ 7,00  
- Couvert artístico: R$ 4,00  
  - **Isento** se o consumo ultrapassar R$ 30,00

O programa deve **ler os dados de um cliente**:

1. Sexo (`F` ou `M`)  
2. Quantidade de cervejas consumidas  
3. Quantidade de refrigerantes consumidos  
4. Quantidade de espetinhos consumidos  

E gerar um **relatório** com:

- Consumo total  
- Couvert artístico (ou isento)  
- Valor do ingresso  
- Valor total a pagar
